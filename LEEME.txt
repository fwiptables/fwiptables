 Notas de fwiptables :

################################################## [only_output]

     Con [only_output] las reglas del firewall son mas utiles

##################################################  [vpn]

     Host [vpn] excepcionales tienen conexion permitida en el firewall

##################################################  [blacklist]

     Host [blacklist] excepcionales tienen conexion denegada en el firewall

##################################################  [uid y/o gid excepcional]

     Usuario y/o grupo excepcional tienen conexion de salida permitida en el firewall

##################################################  [Dos iptables netftilter]

     iptables-legacy   es    soporte para xtables 
     iptables-nft      es    soporte para neftables 

##################################################  [Puertos cliente udp necesarios]

     necesario para actualizar la fecha :   ntp
     necesario para dhcp y conseguir ip :   bootpc
     necesario para resolver dominios   :   domain

##################################################  [Protocolos necesarios para ipv6]

     Necesario protocolo: ipv6-icmp
     Ipv6 funciona tambien con: ipv4 procolos
